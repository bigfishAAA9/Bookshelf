# Bookshelf

一个包含生物信息学与基因组学技术书籍和学习资料的数字书架项目。

> ⚠️ **仅供学习，24小时内自行删除**

## 📚 书籍目录

### Genomics (基因组学)

#### [An Introduction to Statistical Genetic Data Analysis](Genomics/An Introduction to Statistical Genetic Data Analysis/)

**作者**: Melinda C. Mills, Nicola Barban, Felix C. Tropf  
**出版社**: The MIT Press (2020)  
**ISBN**: 9780262538381

本书介绍了统计遗传数据分析的基本概念、方法和应用，适合没有深厚分子生物学或遗传学背景的学生和研究人员。

---

#### [Bioinformatics with Python Cookbook](Genomics/Bioinformatics with Python Cookbook/)

**作者**: Tiago Antao  
**出版社**: Packt Publishing (2018)  
**ISBN**: 9781787122083

本书通过实用的Python代码示例，介绍了生物信息学分析的核心技术，包括下一代测序数据分析、基因组操作、群体遗传学、系统发育分析等。

---

#### [OReilly Bioinformatics Data Skills](Genomics/OReilly Bioinformatics Data Skills Reproducible and Robust Research with Open Source Tools/)

**作者**: Vince Buffalo  
**出版社**: O'Reilly Media (2015)  
**ISBN**: 9781449367371

本书教读者如何使用开源工具进行可重现和稳健的生物信息学研究，涵盖Unix shell、R语言、Git版本控制、数据工具等实用技能。

---

## 🎯 学习目标

### An Introduction to Statistical Genetic Data Analysis
- 理解人类基因组的基本术语和组织结构
- 掌握遗传数据分析的统计方法
- 应用全基因组关联研究(GWAS)技术
- 创建和验证多基因评分
- 理解基因-环境交互作用

### Bioinformatics with Python Cookbook
- 使用Python处理生物信息学数据
- 分析下一代测序数据
- 进行群体遗传学和系统发育分析
- 处理大规模基因组数据集

### OReilly Bioinformatics Data Skills
- 掌握Unix Shell和命令行工具
- 使用R语言进行统计分析和可视化
- 应用Git进行版本控制
- 进行可重现的生物信息学研究

## 📝 如何使用

1. **系统学习**: 按书籍顺序阅读，从基础概念开始
2. **实践操作**: 运行每章提供的代码示例
3. **深入理解**: 完成每章末尾的练习题
4. **扩展阅读**: 参考每章的"Further reading"部分
5. **交叉学习**: 结合三本书的内容，构建完整的生物信息学知识体系
