# Bookshelf

生物信息学与基因组学学习资源库

## 目录结构

```
Bookshelf/
└── Genomics/
    ├── An Introduction to Statistical Genetic Data Analysis/
    ├── Bioinformatics with Python Cookbook/
    └── OReilly Bioinformatics Data Skills/
```

## 书籍列表

### 1. An Introduction to Statistical Genetic Data Analysis

统计遗传数据分析导论，包含15个章节及附录。

### 2. Bioinformatics with Python Cookbook

使用Python进行生物信息学分析的实践指南，包含11个章节。

### 3. OReilly Bioinformatics Data Skills

开源工具实现可重现和稳健研究的数据技能，包含多个章节。
